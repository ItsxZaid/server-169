# Discord Bot (Server 169)

Set up your environment variables in .env file.
```environment
DISCORD_TOKEN=
DATABASE_URL=
```

And then Download Node.js from https://nodejs.org/en/download/

And then Install dependencies with npm install

And then Run the bot with npm run build

And then Run the bot with npm run start

it will show a message in the console that the bot is ready.

if not it mean there was a error in the code or the environment variables are not set correctly.
